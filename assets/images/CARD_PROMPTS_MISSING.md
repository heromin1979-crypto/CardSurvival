## 나노 바나나 사용법

1. 위 **공통 스타일 접두어**를 복사
2. 해당 카드의 프롬프트를 이어 붙여넣기
3. 512×512 정방형으로 생성
4. 파일명과 경로에 맞게 저장
5. `CardFactory.js`의 `CARD_IMAGES`에 매핑 추가

```javascript
// CardFactory.js CARD_IMAGES에 추가 예시
acorn:         'assets/images/food/acorn.png',
acorn_flour:   'assets/images/food/acorn_flour.png',
// ...
```
