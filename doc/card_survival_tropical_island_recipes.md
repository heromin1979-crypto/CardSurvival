## 1. 기초 재료 가공

### 1-1. 돌 가공

| 입력 | 출력 | 시간 |
|------|------|:----:|
| 돌 + 돌 (또는 무거운 돌) | **날카로운 돌** (Sharp Stone) | 1시간 |

> 날카로운 돌은 게임 최초의 절단 도구. 부싯돌(Flint)은 고지대/정동석에서 발견.

### 1-2. 코코넛 가공 체인

| 단계 | 입력 | 출력 | 시간 |
|:----:|------|------|:----:|
| 1 | 코코넛 + 돌/망치 도구 | 껍질 벗긴 코코넛 + 코코넛 껍질 | 30분 |
| 1' | 코코넛 + 절단 도구 | 껍질 벗긴 코코넛 + 코코넛 껍질 | 15분 |
| 2 | 껍질 벗긴 코코넛 + 날카로운 돌 | 코코넛 과육 + 코코넛 워터 + 코코넛 쉘 | — |
| 3 | 코코넛 과육 + 빈 코코넛 쉘 | 코코넛 밀크 (반분량 ×2) | — |
| 4 | 코코넛 밀크 + 불 (끓이기) | **코코넛 오일** | — |
| — | 코코넛 껍질 | 섬유(Fiber) ×2 | 15분 |

### 1-3. 섬유 → 밧줄 가공 체인

| 단계 | 입력 | 출력 |
|:----:|------|------|
| 1 | 섬유(Fiber) ×2 | 섬유 끈 (Fiber Cord) |
| 2 | 섬유 끈 ×2 | 실 (Yarn) |
| 3 | 실 ×2 | 밧줄 (Rope) |

> 섬유 원료: 코코넛 껍질, 뱀풀(Snake Grass)

### 1-4. 나무 가공

| 입력 | 출력 | 비고 |
|------|------|------|
| 돌도끼/도끼 + 나무/장소 | 나무, 막대, 긴 막대 | — |
| 긴 막대 + 칼 | 거친 창 (Rough Spear) | 1시간 |
| 긴 막대 + 모닥불 | 거친 창 (불에 단련) | 45분 |
| 나무 + 절단 도구 | 톱밥 (Sawdust) | 30분 |

### 1-5. 진흙/점토 가공

| 입력 | 출력 | 비고 |
|------|------|------|
| 진흙 + 모래 | 진흙 벽돌 (Mud Brick) | 가마 블루프린트 해금 |
| 점토 → 성형 | 미소성 그릇/항아리/냄비 | — |
| 미소성 용기 + 가마 (3시간) | 점토 그릇/항아리/냄비 | — |

### 1-6. 설탕 생산

| 단계 | 과정 |
|:----:|------|
| 1 | 니파 야자 절단 → 용기 배치 → **니파 수액** 수집 |
| 2 | 니파 수액 + 조리 시설에서 가열 → **설탕** (수액 1그릇 = 설탕 2개) |

### 1-7. 비누 제작

| 단계 | 입력 | 출력 |
|:----:|------|------|
| 1 | 재(Ash) + 물 → 끓이기 | 잿물 (Lye) |
| 2 | 잿물 + 지방(Fat) → 끓이기 | **비누** (Soap) |

### 1-8. 접착제 제작

| 입력 | 출력 |
|------|------|
| 조리 냄비 + 물 + 가죽 + 소석회 → 끓이기 | **접착제** (Glue) |

## 2. 도구 제작

### 2-1. 기본 도구

| 도구 | 재료 | 비고 |
|------|------|------|
| 날카로운 돌 (Sharp Stone) | 돌 + 돌 | 최초 절단 도구 |
| 돌도끼 (Stone Axe) | 돌 ×2 + 막대 + 나무 | 첫 번째 도끼 |
| 손 드릴 (Hand Drill) | 날카로운 돌 + 막대 | 불 피우기 도구 |
| 뼈 칼 (Bone Knife) | 뼈 (동물에서 획득) | "날카로운 도구" 분류 아님 |
| 뼈 바늘 (Bone Needle) | 새 뼈 또는 뼈 파편 | 봉합/바느질용 |
| 뼈 낚시바늘 (Bone Hook) | 새 뼈 | 낚시용 |
| 삽 (Shovel) | 블루프린트 제작 | 우물 해금 |

### 2-2. 복합 도구

| 도구 | 재료 | 비고 |
|------|------|------|
| 부싯돌 창 (Flint Spear) | 긴 막대 + 날카로운 돌 + 밧줄 | 사냥 무기 |
| 낚싯대 (Fishing Rod) | 긴 막대 + 실/줄 + 낚시바늘 | 낚시용 |
| 활 드릴 (Bow Drill) | 제작 숙련도 50%에서 해금 | 빠른 발화 (16분 vs 30분) |
| 실 꿴 바늘 (Threaded Needle) | 뼈 바늘 + 섬유/줄 | 상처 봉합, 의류 수선 |

### 2-3. 구리 도구 (용광로/고급 가마 + 진흙 주형 필요)

| 도구 | 주형 재료 | 비고 |
|------|-----------|------|
| 구리 칼 (Copper Knife) | 진흙 + 경화제 + 구리 ×2 | 무한 수리 가능; "날카로운" 분류 |
| 구리 도끼 (Copper Axe) | 진흙 + 경화제 + 구리 ×5 | 강력, 수리 가능 |
| 구리 삽 (Copper Shovel) | 진흙 + 경화제 + 구리 ×5 | 고급 발굴 |
| 구리 낚시바늘 (Copper Hook) | 구리 | 내구도 우수 |

## 3. 불 관련 레시피

### 3-1. 불 피우기

| 방법 | 시간 |
|------|:----:|
| 손 드릴 + 섬유/마른 잎 → 불씨(Lit Tinder) | 30분 |
| 활 드릴 + 섬유/마른 잎 → 불씨 | 16분 |
| 벌 연무기(Bee Smoker) | 불씨 1개로 16 시간단위 |
| 불씨(Embers) — 숯이 가득 찬 화덕에서 채취 | 6 시간단위 지속 |

### 3-2. 화덕 & 화로

| 구조물 | 재료 | 비고 |
|--------|------|------|
| 모닥불 (Campfire) | 돌 ×4 + 막대 + 나무 | 조리 슬롯 2개; 불씨 필요 |
| 스토브 (Stove) | 블루프린트 | 정수 필터 해금 |
| 가마 (Kiln) | 진흙 벽돌 ×20 + 막대 ×2 | 점토 소성; 점토 용기 레시피 해금 |
| 용광로 (Forge) | 블루프린트 | 구리 제련; 고급 가마보다 저렴 |
| 고급 가마 (Advanced Kiln) | 블루프린트 | 광석 제련; 공간 더 넓음 |

### 3-3. 조명

| 아이템 | 재료 | 내구도 소모 | 특수 |
|--------|------|:----------:|------|
| 횃불 (Torch) | 막대 + 천 + 지방/오일 | -6% / 15분 | 열 제공 |
| 양초 (Candle) | 지방 ×3 + 줄 ×3 | -3% / 15분 | 열 없음 |
| 시트로넬라 양초 | 양초 변형 + 시트로넬라 | -3% / 15분 | 벌레 퇴치 |
| 자스민 양초 | 양초 변형 + 자스민 | -3% / 15분 | 안락 증가 |

### 3-4. 불 부산물

| 입력 | 출력 |
|------|------|
| 꺼진 모닥불 수확 | 숯(Charcoal) + 재(Ash) ×2 |

## 5. 음식 조리

### 5-1. 단순 조리 (재료를 불 위에 드래그)

| 입력 | 출력 |
|------|------|
| 소라 + 불 | 구운 소라 |
| 게 + 불 | 구운 게 |
| 새우 + 불 | 구운 새우 |
| 굴 + 불 | 구운 굴 |
| 고기 + 불 | 구운 고기 (종류별) |
| 달걀 + 불 | 구운 달걀 |
| 참마 + 냄비 + 물 + 불 | 삶은 참마 |

### 5-2. 조리 레시피 (조리 냄비 필요)

| 요리 | 재료 | 비고 |
|------|------|------|
| 아일랜드 치킨 | 새고기 + 코코넛 밀크 + 채소(뱀풀) | 초반 최고 요리 |
| 아일랜드 피시 | 생선 + 코코넛 밀크 + 채소(뱀풀) | 생선 버전 |
| 해물 캐서롤 | 조개 + 굴 + 성게 | 초반 해산물 요리 |
| 달걀 볶음밥 | 쌀 + 달걀 | 쌀은 부패하지 않음 |
| 염소 스튜 | 염소고기 + 각종 재료 | 후반기 요리 |
| 굴 버터 구이 | 굴 + 버터/지방 | 고급 요리 |

### 5-3. 레시피 재료 분류

| 분류 | 해당 아이템 |
|------|------------|
| 채소/샐러드 | 바나나 줄기, 차이나 로즈, 레몬그라스, 뱀풀, 해초 |
| 토핑 | 바나나, 코코넛 과육, 망고, 니파 씨, 버섯류, 구운 아몬드 |
| 필러(탄수화물) | 삶은 참마, 쌀, 사고 가루, 버섯류 |
| 주류 | 미드(Mead), 야자주(Palm Wine) |

### 5-4. 음식 보존

| 방법 | 과정 | 비고 |
|------|------|------|
| 염장 (Salting) | 고기/생선 + 소금 | 부패 없음; 과다 섭취 시 염분 질환 |
| 훈제 (Smoking) | 고기 + 훈제 선반/불 | 부패 25~30%에서 신선도 리셋 |
| 건조 (Drying) | 식물류 + 건조대 | 건기에만 가능; 비 오면 부패 |
| 건조대 (Drying Rack) | 긴 막대 + 줄 | 4칸; 해안/초원 최적 |

## 6. 의약 & 치료

### 6-1. 알로에 베라

| 과정 | 결과 | 비고 |
|------|------|------|
| 알로에 식물 수확 | 알로에 잎 (식물당 최대 6회) | 15분, 1 TP |
| 알로에 잎 직접 적용 | 통증↓, 벌레물림↓, 세균감염률↓, 상처/화상/일광화상 치유 | 잎 1개당 3회 사용 |
| 알로에 잎 가공 | **알로에 젤** (AloeVeraGelProtection +45) | — |

### 6-2. 상처 치료

| 아이템 | 재료 / 사용법 | 효과 |
|--------|-------------|------|
| 실 꿴 바늘 | 뼈 바늘 + 섬유 → 열상(Laceration)에 드래그 | 봉합 |
| 재 드레싱 (Ash Dressing) | 천 + 재 | 상처 세척/보호 |
| 지혈대 (Tourniquet) | 천 기반 | 출혈 정지 |

### 6-3. 비누 (위생/상처 세척)

> 1-7절 참조: 재 + 물 → 잿물 → 잿물 + 지방 → 비누

### 7-3. 고급 의류/방어구

|--------|------|

### 8-3. 침구

|--------|------|
| 나뭇잎 침대 (Leaf Bed) | 기본; 허리 통증 유발 |
| 소박한 침대 (Rustic Bed) | 나뭇잎보다 우수 |
| 침낭 (Bedroll) | 휴대 가능 |
| 해먹 (Hammock) | 허리 통증 없음; 안락 보너스 없음 |
| 나무 침대 (Wooden Bed) | 허리 통증 최소; 안락 제공 |

## 9. 사냥 & 낚시 도구

### 9-1. 낚시 장비

| 도구 | 재료 | 비고 |
|------|------|------|
| 낚싯대 (Fishing Rod) | 긴 막대 + 줄 + 낚시바늘 | 미끼 필요 |
| 뼈 낚시바늘 | 새 뼈 (가공) | 기본 |
| 구리 낚시바늘 | 구리 | 내구도 우수 |
| 물고기 덫 (Fish Trap) | 블루프린트 | 미끼 불필요; 자동; 폭풍에 날아감 |

### 9-2. 덫

| 덫 | 재료 | 해금 조건 | 포획 대상 |
|-----|------|-----------|-----------|
| 낙석 덫 (Deadfall Trap) | 무거운 돌 + 막대 + 날카로운 돌/칼 | 무거운 돌 발견 | 소형 동물 |
| 올무 (Snare Trap) | 막대 ×2 + 섬유끈 ×2 + 절단 도구 | 포획 스킬 7% | 소~중형; 2 TP 설치 |
| 통나무 덫 (Log Trap) | 고급 블루프린트 | 높은 포획 스킬 | 대형 (원숭이, 왕도마뱀, 멧돼지) |
| 함정 구덩이 (Trapping Pit) | 돌 필요; 노동 집약적 | 높은 포획 스킬 | 대형; 생포 가능 (멧돼지, 염소) |

### 9-3. 미끼 팁

- 오래 가는 미끼 사용 (씨앗, 염장 생선)
- **염장 생선** = 최적 미끼 (부패 없음, 포획률 최고)
- 미끼 종류에 따라 초식·육식·잡식 동물 유인

### 9-4. 사냥 무기

| 무기 | 재료 |
|------|------|
| 거친 창 (Rough Spear) | 긴 막대 + 칼 (또는 모닥불) |
| 부싯돌 창 (Flint Spear) | 긴 막대 + 날카로운 돌 + 밧줄 |
| 작살 낚시 (Spearfishing) | 창 + 수중 장소 | 고스킬 시 낚싯대보다 우수 |

## 11. 구리 제련 & 금속 가공

### 11-1. 가공 체인

| 단계 | 과정 |
|:----:|------|
| 1 | 구리 광석 채굴 (고지대, 정동석, 어두운 동굴) |
| 2 | 구리 광석 + 용광로/고급 가마 → **구리 덩어리** (4시간) |
| 3 | 구리 덩어리 + 진흙 주형 → 도구 주조 |

### 11-2. 구리 매장량 (섬 전체 약 9개)

| 위치 | 수량 |
|------|:----:|
| 동부 고지대 무너진 터널 | ~7 |
| 어두운 동굴 | 1 |
| 고지대 정동석 | 가변 |

### 11-3. 재활용

- 구리 도구/장식품 → 용광로/가마에서 녹임 → **구리 금속** → 무한 재주조 (금속 가공 숙련도 상승)

## 12. 뗏목 & 탈출

### 12-1. 뗏목 건조 해금

- **제작 숙련도 50% + 의지력(Determination) 100%** 필요
- 핵심 재료: 통나무, 섬유 끈, 각종 재료

### 12-2. 뗏목 업그레이드

| 업그레이드 | 비고 |
|-----------|------|
| 뗏목 확장 (×2회 가능) | 공간 추가 |
| 가죽 바닥 | 안락 향상 |
| 점토 화로 | 뗏목 위 조리 가능 |
| 보급 상자 | 폭풍 시 아이템 보호 |
| 뗏목 쉘터 | 날씨 보호 |
| 물고기 덫 | 항해 중 식량 |
| 돛 (Sail) | 순풍에 올림; 폭풍에 내림 |

### 12-3. 항해 권장 물품

- 염장/건조 고기 및 생선
- 물 다량 비축
- 보급 상자 1~2개

## 13. 가축 & 축산

### 13-1. 우리 시스템

- **우리 (Enclosure)**: 나무 + 진흙으로 건설; 포획 동물 수용

### 13-2. 가축 종류

| 동물 | 포획 방법 | 생산물 |
|------|-----------|--------|
| 염소 (Goat) | 함정 구덩이 + 참마 미끼; 운반용 썰매로 이동 | 우유 (수유 중 암컷), 고기, 가죽; 새끼 1~2 번식 |
| 멧돼지/돼지 (Boar/Pig) | 함정 구덩이 | 고기, 지방, 가죽 |
| 자고새 (Partridge) | 올무 + 미끼 | 달걀, 고기; 암컷이 부화하는 달걀 생산 |

### 13-3. 사료 구조물

| 구조물 | 비고 |
|--------|------|
| 염소 사료통 / 물통 | 자동 급식 |
| 멧돼지 사료통 | 분해 시 나무 ×6 + 섬유끈 ×4 반환 |

## 14. 장식 & 안락 아이템

| 아이템 | 안락 | 획득 방법 |
|--------|:----:|-----------|
| 나무 조각상 (남자, 여자, 정령, 갈매기, 염소) | 0.75% 각 (최대 6개) | 나무로 제작 |
| 산호 (Coral) | 0.75% 각 | 수영 중 발견 |
| 방해석 결정 (Calcite Crystal) | 0.75% 각 | 정동석 깨기 |
| 진주 목걸이 (Pearl Necklace) | 안락 보너스 | 섬유끈 ×2 + 진주 ×9 |
| 상어 트로피 (Shark Trophy) | 안락 보너스 | 다이빙 중 상어 처치 (생선으로 유인) |

## 15. 분해 레시피

> 대부분의 제작 아이템은 **분해하여 재료를 회수할 수 없음**. 알려진 예외:

| 아이템 | 분해 결과 | 비고 |
|--------|-----------|------|
| 고철 도끼 (Scrap Axe) | 고철 ×1 (제작 시 ×2 사용) | 재료 손실 있음 |
| 고철 삽 (Scrap Shovel) | 고철 ×1 (제작 시 ×2 사용) | 재료 손실 있음 |
| 멧돼지 사료통 (Boar Feeder) | 나무 ×6 + 섬유끈 ×4 | 전량 회수 |
| 베틀 (Loom) | 섬유끈 ~4~6개 | 6개 반환은 버그일 수 있음 |
| 비옷 / 빗물 수집기 | 분해 → 태양열 증류기로 재조립 가능 (역방향도 가능) | 상호 변환 |
| 구리 도구 전반 | 용광로/가마에서 녹여 구리 금속 회수 → 재주조 | 무한 재활용 |

> **주의**: 오두막 개선품(탁자, 침대, 상어 트로피, 책장, 가죽 바닥, 나무 상자)은 오두막 유형 변경 시 대부분 전량 회수되지 않아 재건축 비용이 크다.
