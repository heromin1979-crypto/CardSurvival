## 5. 핵심 수식

### 제작 성공 확률

```
실패율 = 기본 실패율 - 스킬 레벨 보너스 - 캐릭터 보너스
실패율 *= 사기(Morale) 등급 보정
최소 실패율 = 5%  (스킬 만렙이어도 5% 실패 가능)
```

### 해체 재료 드롭

```
각 재료별 독립 확률 판정 (40~100%)
수확 스킬 보너스: +% 추가 드롭 확률
수확 마스터리: 재료 수량 2배
```

### 다단계 제작

```
총 TP = 모든 단계 TP 합산
각 단계 재료 소비: 'start' 또는 'complete' 시점
단계 자동 진행: 해당 단계 TP 소진 시
필요 도구: 보드 위에 설치되어 있어야 함
```
